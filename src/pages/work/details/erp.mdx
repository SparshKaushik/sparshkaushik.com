<pre>



</pre>
## Overview

The in-house ERP system, "Manthan," developed for VIPS-TC College, is a comprehensive solution designed to streamline various administrative and academic processes for an organization with over 7,000 users. This enterprise resource planning software includes a range of modules tailored to the college's specific needs, such as faculty attendance and leave management, event scheduling with hall booking, and a learning management system (LMS) that supports student attendance, notes sharing, and assignment submissions. Integrated with hardware biometric machines, Manthan ensures accurate faculty attendance tracking. The system is accessible via multiple platforms, including a web app, Android app, and an admin dashboard.
